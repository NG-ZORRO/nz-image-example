Automatically add <link rel="preload"> tags to make the browser load images in a higher priority.

By adding srcset attribute to HD displays(retina screens), the browser will automatically load image which has the most suitable pixel ratio of the device.

The component is friendly to image CDN providers, you can either use the built-in CDN loader or customize it.
